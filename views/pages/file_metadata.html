File Metadata
Microservice

How do you use this service?

Select a file to be uploaded.

* No data is saved.

What does this service do?

This will extract the metadata of any file the you upload.

The metadata will include the name of the file, the type, and the size in bytes.

The data will be returned as JSON object once the file is uploaded.

Example Output

{ "name":"some_file.txt", "type":"text/plain", size: 1714 }
