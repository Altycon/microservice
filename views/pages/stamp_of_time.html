Stamp of Time 
Microservice

How do you use this service?

Enter a date or unix timstamp in the address bar along side the URL.

Click links below to try service. ↓

  • www.sotms/api/2015-12-25☜
  • www.sotms/api/1451001600000☜

Example Output ↓

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

Invalid Date ↓

{"unix":null, "utc":"Invalid Date"}

Nothing provided to the 'api' endpoint. ↓

{"error": "No information provided for request"}

What is this service?

This is a simple date api service for internet developers but anyone can use it if you find it useful!

By using the address bar or url, you can can recieve a few pieces of information about any date added to the url.

The data recieved will be in JSON format.

  • * The Unix timestamp: The time in milliseconds since the Unix Epoch on January 1st, 1970.
  • * The Day of the week
  • * The date reformated to universal format.
  • * And a useless bit that shows the time...usually just 00:00:00.
